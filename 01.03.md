# Getting Better Results - Training Notes

## 1. Summary
Source: https://anthropic.skilljar.com/claude-101/383392

This lesson explains how to improve results with Claude through better prompts, iteration, AI Fluency (4D), and evaluation techniques.

Key themes:


### Common challenges when using AI:

* Too generic → add more context (role, audience, constraints)
* Wrong length → specify exact length
* Wrong format → provide examples or structure
* Incorrect facts → verify, ask for sources
* Wrong tone → explicitly describe tone or give examples [Getting be...er results]



## Iteration mindset:

* First prompt is a starting point, not final output
* Improve by giving specific feedback
* Restart conversation if needed [Getting be...er results]



### AI Fluency (4D Framework):

* Delegation → decide human vs AI tasks
* Description → clearly communicate with AI
* Discernment → evaluate outputs critically
* Diligence → use AI responsibly and ethically [Getting be...er results]



### Using evaluations (evals):

Helps measure how well Claude performs for your tasks
Simple approach:

* Gather examples
* Create prompts
* Compare outputs
* Refine prompts

## 2. Q&A
- Why generic responses? → Lack of context  
- Fix length issues? → Specify constraints  
- Fix format issues? → Provide examples  
- AI Fluency? → Effective collaboration with AI  
- 4D? → Delegation, Description, Discernment, Diligence  
- Evals? → Measure performance  

## 3. Quiz
1. Fix generic response → Add context ✅  
2. Not part of 4D → Detection ✅

## 4. Key Takeaways
- Add context  
- Be explicit (length, format, tone)  
- Iterate prompts  
- Apply 4D framework  
- Use evals for validation
