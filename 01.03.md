## 1. Summary
Source: https://anthropic.skilljar.com/claude-101/383392

This lesson explains how to improve results with Claude through better prompts, iteration, AI Fluency (4D), and evaluation techniques.

- Evals? → Measure performance

## 4. Key Takeaways
- Add context  
- Be explicit (length, format, tone)  
- Iterate prompts  
- Apply 4D framework  
- Use evals for validation
