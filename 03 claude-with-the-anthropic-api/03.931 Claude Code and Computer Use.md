# 2. Q\&A

### Q1: What is Claude Code?

* A terminal-based AI coding assistant integrated into the development workflow.

### Q2: What tasks can Claude Code perform?

* File editing, bug fixing, answering coding questions, and assisting with workflows.

### Q3: What is Computer Use?

* A toolset that allows Claude to interact with desktop environments and applications.

### Q4: How does Computer Use extend Claude’s capabilities?

* It moves beyond text into UI-based interactions (browser, apps, visual navigation).

### Q5: Why are these tools important?

* They demonstrate real-world **AI agent implementations**.

### Q6: What core agent capabilities are illustrated?

* Tool usage, multi-step execution, environment interaction, autonomous reasoning.

* Claude Code = **developer-centric agent (CLI environment)**

* Core agent pattern:
