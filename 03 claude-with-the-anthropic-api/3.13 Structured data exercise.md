# 2. ❓ Q\&A (from page content & context)

### Q1. What is the goal of the structured data exercise?

* To practice generating and controlling **structured outputs from Claude**.

### Q2. What concept is highlighted in this exercise?

* Stop sequences
* Message prefilling

### Q3. What are stop sequences used for?

* To **force the model to stop generating output** when a specific sequence appears.

### Q4. What is message prefilling?

* Providing **partial responses upfront** so Claude continues in a controlled format.

### Q5. Why is structured output important?

* It ensures:
  * Predictable responses
  * Easier parsing (JSON, XML)
  * Integration into applications

# 4. 🔑 Key Takeaways

* **Structured output is critical** for real applications (APIs, automation, pipelines)

  * **Message prefilling**
