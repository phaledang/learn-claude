
# 📘 1. Summary

**Source:** *Being specific* (Anthropic Claude API course)
https://anthropic.skilljar.com/claude-with-the-anthropic-api/287740
This lesson explains that **being specific in prompts significantly improves Claude’s output quality and consistency**. Instead of leaving interpretation open, you provide clear guidance.

### Key concepts:

* Without specificity → outputs vary widely (length, content, structure)
* With specificity → **controlled, predictable, high-quality outputs**

### Two main ways to be specific:

1. **Output Quality Guidelines**
   * Define expectations for the result
   * Control:
     * Length
     * Format/structure
     * Required elements
     * Tone/style
   * Example:
     * “Under 1000 words”
     * “Include supporting character”
     * “Use specific format”

2. **Process Steps**
   * Guide the **thinking process**
   * Force structured reasoning before answering
   * Example workflow:
     * Brainstorm ideas
     * Select best option
     * Outline solution
     * Refine output

### Real impact:

* Adding clear guidelines can **double output quality scores** (example: 3.92 → 7.86)

### When to use:

* ✅ Always → use **output guidelines**
* ✅ Complex cases → add **process steps**
  * Troubleshooting
  * Decision-making
  * Analysis tasks

### Best practice:

* ✅ Combine BOTH approaches for professional prompts  
  → ensures **quality + structured reasoning**

# ❓ 2. Q\&A

### Q1. Why is being specific important when prompting Claude?

**A:** It improves output quality and consistency by reducing ambiguity.

### Q2. What are “output quality guidelines”?

**A:** Instructions that define how the final answer should look (length, format, tone, content).

### Q3. What are “process steps”?

**A:** Step-by-step instructions telling Claude how to think before responding.

### Q4. When should you use process steps?

**A:** For complex tasks like analysis, troubleshooting, or decision-making.

### Q5. What happens if you don’t provide specific instructions?

**A:** Claude may generate unpredictable or inconsistent outputs.
