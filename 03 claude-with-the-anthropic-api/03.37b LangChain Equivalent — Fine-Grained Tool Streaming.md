# LangChain Equivalent — Fine-Grained Tool Streaming

## 1. Concept Mapping

| Claude API concept                  | LangChain equivalent                                          |
| ----------------------------------- | ------------------------------------------------------------- |
| Tool schema                         | `@tool` / StructuredTool                                      |
| Tool calling                        | `bind_tools()`                                                |
| Streaming events (`InputJsonEvent`) | `astream_events()` / callbacks                                |
| `partial_json`                      | incremental chunks in event stream                            |
| `snapshot`                          | accumulated state managed in handler                          |
| `fine_grained=True`                 | **No direct flag** → replicate via streaming + manual parsing |

***

## 2. Define Tool Schema

```python
from langchain.tools import tool

@tool
def save_article(abstract: str, meta: dict) -> str:
    return f"Saved article with {len(abstract)} chars"
```

***

## 3. Bind Tool to Model

```python
from langchain_anthropic import ChatAnthropic

llm = ChatAnthropic(
    model="claude-3-opus-20240229",
    streaming=True
)

llm_with_tools = llm.bind_tools([save_article])
```

***

## 4. Streaming Tool Call (Fine-Grained Behavior)

LangChain does **not buffer top-level JSON like Anthropic API default mode**  
→ streaming is already closer to **fine-grained behavior**

```python
async for event in llm_with_tools.astream_events(
    "Summarize this paper and call save_article tool",
    version="v1"
):
    if event["event"] == "on_tool_start":
        print("Tool started")

    elif event["event"] == "on_tool_stream":
        chunk = event["data"]

        # partial JSON-like data
        print(chunk)

    elif event["event"] == "on_tool_end":
        print("Tool completed")
```

***

## 5. Manual JSON Accumulation (Snapshot Equivalent)

LangChain does **not provide snapshot automatically**  
→ implement accumulation layer

```python
import json

buffer = ""

async for event in llm_with_tools.astream_events(
    "Generate structured article metadata",
    version="v1"
):
    if event["event"] == "on_tool_stream":
        chunk = event["data"]["chunk"]

        buffer += chunk

        try:
            parsed = json.loads(buffer)
            print("Valid JSON snapshot:", parsed)
        except json.JSONDecodeError:
            print("Partial / invalid JSON, continue...")
```

***

## 6. Error Handling (Same as Fine-Grained Requirement)

* No validation from framework
* Must handle invalid JSON

```python
try:
    parsed = json.loads(buffer)
except json.JSONDecodeError:
    # continue streaming
    pass
```

***

## 7. Callback-Based Alternative (Cleaner Design)

```python
from langchain.callbacks.base import AsyncCallbackHandler

class ToolStreamHandler(AsyncCallbackHandler):
    def __init__(self):
        self.buffer = ""

    async def on_tool_new_token(self, token, **kwargs):
        self.buffer += token

        try:
            parsed = json.loads(self.buffer)
            print("Snapshot:", parsed)
        except:
            pass
```

Attach handler:

```python
llm = ChatAnthropic(
    model="claude-3-opus-20240229",
    streaming=True,
    callbacks=[ToolStreamHandler()]
)
```

***

## 8. Key Differences vs Claude Native API

| Feature           | Claude API            | LangChain                        |
| ----------------- | --------------------- | -------------------------------- |
| JSON validation   | Enabled by default    | Not enforced                     |
| Fine-grained mode | `fine_grained=True`   | Default behavior already similar |
| Event type        | `InputJsonEvent`      | `on_tool_stream`, callbacks      |
| Snapshot support  | Built-in (`snapshot`) | Manual implementation            |
| Buffering         | API-level             | Developer-controlled             |

***

## 9. When This Matches Fine-Grained Tool Calling

LangChain approach ≈ fine-grained mode when:

* Streaming enabled
* Using event-based handling (`astream_events`)
* Manual JSON parsing implemented
* No post-validation layer added

***

## 10. Minimal End-to-End Pattern

```python
llm = ChatAnthropic(streaming=True).bind_tools([save_article])

buffer = ""

async for event in llm.astream_events(prompt):
    if event["event"] == "on_tool_stream":
        buffer += event["data"]["chunk"]

        try:
            parsed = json.loads(buffer)
            # process early result
        except:
            pass
```

***

## 11. Key Takeaways

* LangChain **implicitly behaves like fine-grained mode**
* No built-in JSON validation pipeline
* Requires:
  * manual accumulation
  * error handling
* Streaming primitives:
  * `astream()`
  * `astream_events()`
  * callbacks
