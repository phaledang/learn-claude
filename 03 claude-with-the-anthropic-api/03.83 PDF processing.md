**PDF processing**
## 1. Summary of the page

**Source URL:** <https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768>

* Claude supports **direct PDF processing**, allowing reading and analysis of PDF documents as input. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)
* Implementation is **similar to image processing**, with only a few changes in file type and metadata. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)
* Key setup flow:
  * Read PDF as binary
  * Encode to Base64
  * Send as a `document` type in request
  * Add text instruction (e.g., summarization) [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)
* Important differences from image handling:
  * `type`: `"document"` instead of`"image"`
  * `media_type`: `"application/pdf"`
  * Use `.pdf` file and `file_bytes` variable [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

**Capabilities of PDF support:**

* Extract **text content**
* Understand **images and charts**
* Parse **tables and relationships**
* Recognize **document structure and formatting** [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

→ Result: Claude acts as a **unified document analysis engine** for PDFs, supporting summarization, extraction, and interpretation. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

## 2. Q\&A from the page

**Q1: How does Claude process PDFs?**  
A: By encoding the PDF into Base64 and sending it as a `document` input in the message payload. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

**Q2: Is PDF processing different from image processing?**  
A: The workflow is nearly identical, with differences in file type, variable naming, and media type. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

**Q3: What media type is used for PDFs?**  
A: `"application/pdf"`. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

**Q4: What type is used in the message block for PDFs?**  
A: `"document"`. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

**Q5: What can Claude extract from PDF files?**  
A: Text, images, charts, tables, and structural information. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)

**Q6: What kind of tasks can be performed on PDFs?**  
A: Summarization, data analysis, and targeted content extraction. [\[PDF support\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287768)
