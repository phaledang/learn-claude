## 1. Summary

**Source:** Claude Code setup (active page)

* **Claude Code** is a **terminal-based coding assistant** that runs inside the command line, providing direct access to Claude for development tasks. [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)
* Core capabilities:
  * **File operations**: search, read, edit project files
  * **Terminal execution**: run commands within the conversation
  * **Web access**: fetch documentation and examples
  * **MCP server integration**: extend functionality with external tools/services [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)
* **MCP integration** is highlighted as the key extensibility mechanism—allowing integration with databases, APIs, and custom tools. [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)
* Cross-platform support: **MacOS, Windows (WSL), Linux** [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)
* Setup process:
  1. Install Node.js
  2. Install via npm: `npm install -g @anthropic-ai/claude-code`
  3. Run `claude` and log in [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)

## 2. Q\&A from the page

**Q1. What is Claude Code?**  
A terminal-based assistant that enables interacting with Claude directly inside the command line for coding tasks. [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)

**Q2. What tasks can Claude Code perform?**

* File manipulation
* Running terminal commands
* Fetching web content
* Integrating external tools via MCP [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)

**Q3. Why is MCP important in Claude Code?**  
It allows extending capabilities by connecting additional tool servers such as APIs or databases. [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)

**Q4. What environments are supported?**  
MacOS, Windows (via WSL), and Linux. [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)

**Q5. How is Claude Code installed?**  
Using Node.js and npm with a global install command, then launching via terminal. [\[Claude Code setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287788)
