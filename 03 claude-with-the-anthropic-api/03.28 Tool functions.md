### 🔹 Key Concepts

* **Tool functions** are **Python functions** that Claude can call to:
  * Retrieve real-time information
  * Perform actions on behalf of users
* Claude **decides when to call a tool** during a conversation based on user intent. [\[Tool functions\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287756)

### 🔹 Example Use Cases

* Get current date/time
* Add duration to a date
* Set reminders
* Fetch weather or other external data [\[Tool functions\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287756)

### 🔹 How It Works

1. User asks a question
2. Claude determines it needs external data
3. Claude calls the appropriate **tool function**
4. Function executes and returns result
5. Claude uses the result to respond

### 🔹 Example Tool Function

```python
def get_current_datetime(date_format="%Y-%m-%d %H:%M:%S"):
    if not date_format:
        raise ValueError("date_format cannot be empty")
    return datetime.now().strftime(date_format)
```

### 🔹 Best Practices

* ✅ **Use descriptive names**
  * Clear function + parameter naming
* ✅ **Validate inputs**
  * Prevent invalid or empty values
* ✅ **Provide meaningful error messages**
  * Claude can learn and retry with corrected inputs [\[Tool functions\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287756)

### 🔹 Key Insight

* Claude can **learn from errors** → good validation improves accuracy of tool usage. [\[Tool functions\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287756)

# ❓ 2. Q\&A

### Q1. What is a tool function?

**A:** A Python function that Claude can automatically call to get data or perform actions.

### Q2. When does Claude use a tool function?

**A:** When it determines additional external information is needed to answer a user.

### Q3. Why is input validation important?

**A:** It prevents invalid inputs and helps Claude retry correctly when errors occur.

### Q4. How does Claude benefit from error messages?

**A:** It can interpret errors and adjust its next function call accordingly.

### Q5. What is required after defining a function?

**A:** You must create a **JSON schema** and integrate it into the chat system.
