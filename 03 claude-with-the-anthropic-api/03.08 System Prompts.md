### What are System Prompts?

* A **system prompt** is a string that defines **how Claude should behave**.
* It customizes:
  * Tone
  * Style
  * Role
  * Response strategy [\[System prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287733)

### Why they matter

* Without system prompt → generic answers
* With system prompt → **role-specific, controlled behavior**
* Example:
  * Math tutor → gives hints instead of answers
  * Not desired → giving direct solutions immediately [\[System prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287733)

### How System Prompts Work

* Passed as `system` parameter in API call
* Claude adapts responses based on the defined role
* Keeps responses consistent and aligned with intent [\[System prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287733)

```python
system_prompt = """
You are a patient math tutor.
Do not directly answer questions.
Guide students step by step.
"""

client.messages.create(
    model=model,
    messages=messages,
    max_tokens=1000,
    system=system_prompt
)
```

### Key behavior change
```
def chat(messages, system=None):
    params = {
        "model": model,
        "max_tokens": 1000,
        "messages": messages,
    }
    
    if system:
        params["system"] = system
    
    message = client.messages.create(**params)
    return message.content[0].text
```
* Without system prompt → full solution immediately
* With system prompt → asks guiding questions instead [\[System prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287733)
