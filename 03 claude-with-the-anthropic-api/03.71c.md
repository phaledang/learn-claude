# 🔷 1. Claude-native RAG (simple / built-in retrieval)

👉 Use when:

* You want **fast setup**
* Minimal infra (documents handled by Claude)

```python
import anthropic

client = anthropic.Anthropic(api_key="YOUR_API_KEY")

response = client.messages.create(
    model="claude-3-5-sonnet",
    max_tokens=1000,
    messages=[
        {
            "role": "user",
            "content": [
                # Document (can be PDF/text)
                {
                    "type": "document",
                    "source": {
                        "type": "text",
                        "data": open("report.txt").read()
                    }
                },
                # Question
                {
                    "type": "text",
                    "text": "What are the key risk factors?"
                }
            ]
        }
    ]
)

print(response.content)
```

### ✅ Notes

* No explicit retrieval pipeline
* Claude internally uses context + reasoning
* Works best with **medium-size docs / few docs**

# 🔷 2. LangChain RAG (full pipeline control)

👉 Use when:

* You want **custom chunking / retrieval / agents**

```python
from langchain.document_loaders import TextLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter
from langchain.vectorstores import FAISS
from langchain.embeddings import OpenAIEmbeddings
from langchain.chat_models import ChatAnthropic
from langchain.chains import RetrievalQA

# 1. Load documents
loader = TextLoader("report.txt")
docs = loader.load()

# 2. Chunking
splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=100
)
chunks = splitter.split_documents(docs)

# 3. Embedding + Vector store
embeddings = OpenAIEmbeddings()
vectorstore = FAISS.from_documents(chunks, embeddings)

# 4. Retriever
retriever = vectorstore.as_retriever(search_kwargs={"k": 3})

# 5. LLM (Claude)
llm = ChatAnthropic(model="claude-3-5-sonnet")

# 6. RAG chain
qa = RetrievalQA.from_chain_type(
    llm=llm,
    retriever=retriever
)

# 7. Query
response = qa.run("What are the risk factors?")
print(response)
```

### ✅ Notes

* Full control:
  * chunking strategy
  * embedding model
  * retrieval logic
* Replace FAISS → Pinecone / Weaviate / Azure Search
* Easily extend to **agent workflows**

***

# 🔷 3. Azure AI Search RAG (enterprise retrieval)

👉 Use when:

* Large-scale documents
* Need **hybrid search + security + governance**

***

## Step 1: Index + Search (Azure)

```python
from azure.search.documents import SearchClient
from azure.core.credentials import AzureKeyCredential

endpoint = "https://<your-service>.search.windows.net"
index_name = "documents"
api_key = "YOUR_AZURE_KEY"

client = SearchClient(endpoint, index_name, AzureKeyCredential(api_key))

results = client.search(
    search_text="risk factors",
    top=3
)

contexts = []
for r in results:
    contexts.append(r["content"])
```

***

## Step 2: Send to Claude

```python
import anthropic

client = anthropic.Anthropic(api_key="YOUR_API_KEY")

context_text = "\n\n".join(contexts)

response = client.messages.create(
    model="claude-3-5-sonnet",
    max_tokens=1000,
    messages=[
        {
            "role": "user",
            "content": f"""
Use the following context:

{context_text}

Question: What are the risk factors?
"""
        }
    ]
)

print(response.content)
```

***

### ✅ Notes

* Azure handles:
  * indexing
  * embeddings
  * hybrid search (keyword + vector)
* Claude handles:
  * reasoning + answer generation

# 🔷 4. Combined (Recommended Architecture)

👉 Best practice (what you should use in enterprise):

```python
# LangChain + Azure AI Search + Claude

from langchain.retrievers import AzureAISearchRetriever
from langchain.chat_models import ChatAnthropic
from langchain.chains import RetrievalQA

# Azure retriever
retriever = AzureAISearchRetriever(
    service_name="your-service",
    index_name="documents",
    api_key="YOUR_KEY"
)

# Claude LLM
llm = ChatAnthropic(model="claude-3-5-sonnet")

qa = RetrievalQA.from_chain_type(
    llm=llm,
    retriever=retriever
)

response = qa.run("What are the risk factors?")
print(response)
```
