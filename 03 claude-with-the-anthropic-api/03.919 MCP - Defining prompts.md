### Core concept

* **Prompts in MCP (Model Context Protocol)** = **pre-built, reusable instruction templates** defined by server authors. [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)
* Purpose: provide **higher-quality, consistent results** compared to ad-hoc user prompts. [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)

### Why prompts matter

* Users *can* write their own prompts → results vary in quality.
* MCP prompts = **tested + structured + optimized instructions** → better outputs. [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)

### How prompts work

* Defined in MCP server using `@mcp.prompt()`
* Return a **list of messages (user/assistant)** that form a complete prompt
* Clients call prompt → send message directly to Claude [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)

### Example (format prompt)

```python
@mcp.prompt(
    name="format",
    description="Rewrites the contents of the document in Markdown format."
)
def format_document(doc_id: str = Field(description="Id of the document to format")) -> list[base.Message]:
    prompt = f"""
    Your goal is to reformat a document to be written with markdown syntax.
    The id of the document you need to reformat is: {doc_id}
    Add in headers, bullet points, tables, etc as necessary.
    Use the 'edit_document' tool to edit the document.
    """
    return [base.UserMessage(prompt)]
```

→ Prompts can **invoke tools + enforce structure** [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)

### Testing

* Use **MCP Inspector**
* Validate:
  * variable interpolation
  * message structure
  * expected output format [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)

### Best practices

* Focus on **core tasks of MCP server**
* Be **specific, detailed**
* Include **clear descriptions**
* Test with multiple inputs
* Combine prompts with **tools & resources** [\[Defining prompts\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287784)

## 2. Q\&A (training style)

**Q1. What is an MCP prompt?**  
A pre-defined template containing structured instructions returned as messages for Claude.

**Q2. Why use MCP prompts instead of user-written prompts?**  
To ensure higher quality, consistency, and reliability of outputs.

**Q3. What does `@mcp.prompt()` do?**  
Registers a reusable prompt with name and description inside the MCP server.

**Q4. What does a prompt return?**  
A list of `Message` objects (e.g., `UserMessage`) used directly in Claude requests.

**Q5. How do prompts interact with tools?**  
Prompts can explicitly instruct Claude to call tools (e.g., `edit_document`).

**Q6. How are prompts tested?**  
Using the MCP Inspector to validate generated messages and parameters.

**Q7. What makes a good prompt?**  
Detailed instructions, clear purpose, tested, aligned with server capabilities.
