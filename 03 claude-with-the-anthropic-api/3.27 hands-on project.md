

# 📘 1. Summary

**Page:** Project overview  
**URL:** <https://anthropic.skilljar.com/claude-with-the-anthropic-api/287751>

### 🔹 What this page covers

* Introduces a **hands-on project**: building a reminder system using Claude.
* Goal: enable natural language like  
  → *“Set a reminder for my doctor’s appointment. It’s a week from Thursday”*
* Highlights **limitations of Claude** and how to overcome them using tools.

### 🔹 Key challenges identified

* **Limited time awareness** → Claude may know date but not precise time
* **Date calculation issues** → struggles with future date math
* **No built‑in reminder capability** → cannot execute real-world actions

### 🔹 Solution approach

* Use **custom tools** to extend Claude’s capabilities.
* Build 3 tools:
  1. Get current date & time
  2. Add duration to date/time
  3. Set a reminder

### 🔹 Core concept

* Instead of forcing prompts to solve everything → **extend Claude via tools**
* Claude combines tools to handle complex requests.

👉 This is a **foundational pattern for agent design**.

# ❓ 2. Q\&A

### Q1: What is the goal of the project?

**A:** Build a system where Claude can set reminders from natural language input.

### Q2: Why can’t Claude handle reminders directly?

**A:** It lacks built-in capability to execute actions like setting reminders.

### Q3: What is one limitation of Claude with time handling?

**A:** It may struggle with calculating future dates accurately.

### Q4: How do tools help Claude?

**A:** Tools extend Claude’s capabilities to perform actions and computations it cannot do natively.

### Q5: What are the three tools introduced?

**A:**

* Get current date/time
* Add duration to date/time
* Set a reminder

### Q6: What principle does this project demonstrate?

**A:** Extend AI capabilities using tools instead of relying only on prompts.
