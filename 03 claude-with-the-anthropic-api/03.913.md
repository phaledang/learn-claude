This page introduces a **hands-on MCP (Model Context Protocol) project** that builds a **CLI-based chatbot system** to demonstrate how MCP clients and servers interact. [\[Project setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287785)

### Core Concepts

* The system has **two main components**:
  * **MCP Client** Ôćĺ handles user interaction (CLI chatbot)
  * **MCP Server** Ôćĺ manages document operations (read/write) [\[Project setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287785)
* The server exposes **two tools**:
  * Read document content
  * Update document content [\[Project setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287785)
* Documents are stored **in memory (no database)** for simplicity [\[Project setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287785)

### Architecture Insight

* In real-world scenarios:
  * Usually build **either a client OR a server**
  * Not both in the same project
* This project builds both **for learning communication flow** [\[Project setup\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287785)

### Setup Workflow

* Download project (`cli_project.zip`)
* Add Anthropic API key to `.env`
* Install dependencies (`uv` recommended or `pip`)
* Run app:

```bash
uv run main.py
# or
python main.py
```
