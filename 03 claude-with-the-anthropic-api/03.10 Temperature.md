# 🔗 Summary

**Source:** [Temperature – Anthropic Course](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

## 📌 Key Concepts

* **Temperature** controls how **predictable vs creative** Claude’s output is. [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)
* Value range: **0 → 1**
  * **0.0 → deterministic (predictable)**
  * **1.0 → creative (randomness increases)** [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

## 🧠 How Claude Generates Text

1. **Tokenization** – split input into tokens
2. **Prediction** – assign probabilities to next token
3. **Sampling** – pick next token based on probabilities [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

👉 Temperature affects **Step 3 (sampling)**.

## 🎛️ Temperature Behavior

* **Low temperature**
  * Picks highest-probability token
  * Very consistent output
* **High temperature**
  * Flattens probabilities
  * More diverse, creative output [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

## 📊 Recommended Ranges

| Temperature   | Use Cases                                         |                                                             |
| ------------- | ------------------------------------------------- | ----------------------------------------------------------- |
| **0.0 – 0.3** | Factual Q\&A, coding, data extraction, moderation |                                                             |
| **0.4 – 0.7** | Summaries, learning content, reasoning            |                                                             |
| **0.8 – 1.0** | Brainstorming, creative writing, marketing ideas  |  [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728) |

## 💻 Code Example

```python
def chat(messages, system=None, temperature=1.0):
    params = {
        "model": model,
        "max_tokens": 1000,
        "messages": messages,
        "temperature": temperature
    }
    if system:
        params["system"] = system

    message = client.messages.create(**params)
    return message.content[0].text
```

# ❓ Q\&A

### 1. What does temperature control?

* It controls how **predictable or creative** the response is. [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

### 2. What happens at temperature = 0?

* Model becomes **fully deterministic** (always chooses highest probability token). [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

### 3. What happens at temperature = 1?

* Distribution becomes more even → **more randomness and creativity**. [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

### 4. Does high temperature guarantee different outputs?

* No. It only **increases the probability of variation**, not guaranteed diversity. [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)

### 5. Which temperature should you use for coding tasks?

* **Low temperature (0.0–0.3)** [\[Temperature\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287728)
