# 📘 1. Summary

**Page:** *Making a request*  
**URL:** <https://anthropic.skilljar.com/claude-with-the-anthropic-api/287725>

### 🔑 Key points

* Explains how to **send your first request to the Claude API (Python)**. [\[Making a request\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287725)
* Core setup steps:
  * Install dependencies: `anthropic`, `python-dotenv` [\[Making a request\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287725)
  * Store API key in `.env` for security [\[Making a request\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287725)
* Create API client:
  ```python
  from anthropic import Anthropic
  client = Anthropic()
  ```
* Main API method:
  ```python
  client.messages.create()
  ```
* 3 required parameters:
  * `model` → Claude model to use
  * `max_tokens` → response length limit (safety cap)
  * `messages` → conversation history [\[Making a request\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287725)
* Message structure:
  * `role`: `"user"` or`"assistant"`
  * `content`: text input/output [\[Making a request\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287725)
* Example request:
  ```python
  message = client.messages.create(
      model="claude-sonnet-4-0",
      max_tokens=1000,
      messages=[{"role": "user", "content": "What is quantum computing?"}]
  )
  ```
* Extract response:
  ```python
  message.content[0].text
  ```

# ❓ 2. Q\&A

### Basic understanding

* **Q: What is the main function to call Claude API?**  
  A: `client.messages.create()`

* **Q: What are the 3 required parameters?**  
  A: `model`, `max_tokens`, `messages`

* **Q: What does `max_tokens` do?**  
  A: Limits response length (safety cap, not a target)

* **Q: What are the message roles?**  
  A: `user` and `assistant`

***

### Practical usage

* **Q: Why use `.env` file?**  
  A: To securely store API keys and avoid committing them to source control

* **Q: How do you extract Claude’s response?**  
  A: `message.content[0].text`

* **Q: What does the messages field represent?**  
  A: The conversation history like a chat
