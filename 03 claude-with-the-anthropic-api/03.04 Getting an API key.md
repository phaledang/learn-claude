## 1️⃣ Summary of the Page

* This page explains **how to create an Anthropic API key** required to call the Claude API. [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)
* The API key:
  * Is **secret and must be stored securely**
  * Is required for **making API requests in the next steps of the course** [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

### High-level flow:

1. Go to Anthropic API Console
2. Navigate to API key section
3. Create a new key
4. Assign workspace + name
5. Copy and securely store the key (only shown once) [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

***

## 2️⃣ Step-by-step Process (from page)

* Navigate to: [Anthropic Console](https://console.anthropic.com/) [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)
* Click**“Get API Keys”** button
* Click**“Create Key”**
* Provide:
  * Workspace (e.g., *Default*)
  * Key name (e.g., *Anthropic Course*)
* Copy the generated key immediately
  * ⚠️ Only shown once → regenerate if lost [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

## 3️⃣ Q\&A (from content)

**Q1: Why do you need an API key?**

* To authenticate and make requests to the Claude API. [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

**Q2: Where do you create the API key?**

* In the Anthropic API Console at the dashboard. [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

**Q3: What information is required when creating a key?**

* Workspace and a key name. [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

**Q4: Can you view the API key again later?**

* No, it is only displayed once. [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)

**Q5: What should you do if you lose the key?**

* Delete the old key and generate a new one. [\[Getting an API key\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/296766)
