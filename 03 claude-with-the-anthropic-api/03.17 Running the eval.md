# 📘 1. Summary

**Page:** *Running the eval*  
**URL:** <https://anthropic.skilljar.com/claude-with-the-anthropic-api/287743>

### 🎯 Purpose

This lesson explains how to **build and run an evaluation (eval) pipeline** for Claude using a dataset of test cases.

### 🔑 Core Flow

The evaluation workflow follows a clear sequence:

1. Take a **dataset of test cases**
2. Merge each test case with a **prompt template**
3. Send it to **Claude**
4. **Evaluate (grade)** the response

### 🧱 Core Components (Functions)

#### 1. `run_prompt`

* Combines test case input into a prompt
* Sends request to Claude
* Returns response output  
  👉 Current version uses a **simple prompt** (no formatting rules)

#### 2. `run_test_case`

* Calls `run_prompt`
* Assigns a **score (currently hardcoded)**
* Returns:
  * output
  * test\_case
  * score

💡 Note: Real grading logic is not implemented yet

#### 3. `run_eval`

* Iterates through the dataset
* Runs each test case
* Collects results into a list

***

### ▶️ Running the Eval

* Load dataset from `dataset.json`
* Execute `run_eval(dataset)`
* Processing may take \~30 seconds depending on dataset size

***

### 📊 Output Format

Each result returned contains:

* `output`: Claude’s response
* `test_case`: input data
* `score`: evaluation score

***

### ⚠️ Key Observations

* Output is **verbose** due to lack of prompt constraints
* Current scoring is **placeholder → not meaningful yet**
* Pipeline is **simple but foundational**

  * input → model → output → scoring pipeline

# ❓ 2. Q\&A

### Q1. What is the purpose of an eval pipeline?

**A:** To test how well Claude performs across multiple test cases by generating and grading outputs.

### Q2. What does `run_prompt` do?

**A:** It prepares the prompt and sends it to Claude, returning the response.

### Q3. Why is the score currently not useful?

**A:** Because it is hardcoded instead of calculated using evaluation logic.

### Q4. What does `run_eval` return?

**A:** A list of results containing output, test case, and score.

### Q5. Why are responses verbose?

**A:** Because no formatting instructions are included in the prompt.
