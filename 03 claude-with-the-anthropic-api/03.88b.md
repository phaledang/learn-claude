# LangChain / LangGraph equivalent architecture + code

## File upload + sandbox execution pattern

The source page describes a pattern where a file is uploaded first, then referenced during analysis, and Python code is executed in an isolated environment to generate outputs such as plots or reports. [\[bing.com\]](https://bing.com/search?q=LangGraph+persistence+docs)

A close public-framework equivalent is:

* **LangChain** for tool-based agent execution and file loading. [\[reference....gchain.com\]](https://reference.langchain.com/python/langchain-community/document_loaders), [\[pypi.org\]](https://pypi.org/project/langchain-sandbox/), [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/langgraph/8.1-react-agent-%28create_react_agent%29)
* **LangGraph** for stateful orchestration, retries, and checkpointed execution. [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/docs/2.2-langgraph-framework-documentation), [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/langgraph/8.1-react-agent-%28create_react_agent%29)
* **Sandboxed runtime** for isolated code execution. [\[pypi.org\]](https://pypi.org/project/langchain-sandbox/)
* **Object/file storage** for uploaded inputs and generated output artifacts. This storage pattern is an implementation suggestion, not a claim from the cited docs.

***

# 1) Conceptual mapping

| Claude page concept                      | LangChain / LangGraph equivalent                     |
| ---------------------------------------- | ---------------------------------------------------- |
| Files API upload                         | File/object storage + loader step                    |
| `file_id` reference                      | file path / file URI / storage key                   |
| Code execution tool                      | sandbox execution tool                               |
| Multi-step execution in one conversation | agent loop / graph node loop                         |
| Download generated artifacts             | persist output files and return their paths or links |
| Conversation continuity                  | LangGraph checkpoints with `thread_id`               |

The source page explicitly describes uploaded file reuse, isolated execution, multiple execution cycles, and downloadable generated files.\
LangGraph persistence explicitly describes checkpointed state organized into threads. [\[bing.com\]](https://bing.com/search?q=LangGraph+persistence+docs) [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/docs/2.2-langgraph-framework-documentation)

# 2) Recommended architecture

## Option A — LangChain agent + external sandbox

**Use when:**

* orchestration is simple
* most requests are single-turn or short-lived
* retries and durable state are handled outside the agent

### Flow

1. Upload file to controlled storage.
2. Register metadata (job ID, file path, MIME type).
3. Pass file metadata and task prompt to a LangChain agent.
4. Agent can inspect the file, decide whether code execution is needed, and call a sandbox tool.
5. Sandbox generates outputs.
6. Outputs are written back to storage.
7. Agent returns summary + artifact locations.

LangChain document loaders provide a standard interface for loading files into `Document` objects.\
LangChain sandbox documentation describes isolated environments used to execute code and interact with filesystems. [\[reference....gchain.com\]](https://reference.langchain.com/python/langchain-community/document_loaders) [\[pypi.org\]](https://pypi.org/project/langchain-sandbox/)

## Option B — LangGraph workflow + sandbox + checkpointing

**Use when:**

* processing is multi-step
* failures must be resumable
* long-running jobs are common
* human approval / review may be added later
* thread-level memory is required

### Flow

1. `ingest_file`
2. `inspect_file`
3. `plan_analysis`
4. `execute_in_sandbox`
5. `persist_outputs`
6. `summarize_results`
7. `complete`

LangGraph persistence saves graph state as checkpoints and supports memory, time travel, and fault-tolerant execution. [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/docs/2.2-langgraph-framework-documentation)

# 3) LangChain code sample

```python
from __future__ import annotations

import os
import json
import uuid
import shutil
import tempfile
from dataclasses import dataclass
from typing import Optional, Dict, Any

from langchain.tools import tool
from langchain.agents import create_agent
from langchain_anthropic import ChatAnthropic

from langchain_community.document_loaders import CSVLoader, PyPDFLoader, TextLoader


@dataclass
class StoredFile:
    job_id: str
    file_path: str
    original_name: str
    mime_type: str


class LocalObjectStore:
    def __init__(self, root: str = "./object_store"):
        self.root = root
        os.makedirs(self.root, exist_ok=True)

    def put_file(self, src_path: str, original_name: str, mime_type: str) -> StoredFile:
        job_id = str(uuid.uuid4())
        job_dir = os.path.join(self.root, job_id)
        os.makedirs(job_dir, exist_ok=True)

        dst_path = os.path.join(job_dir, original_name)
        shutil.copy2(src_path, dst_path)

        return StoredFile(
            job_id=job_id,
            file_path=dst_path,
            original_name=original_name,
            mime_type=mime_type
        )

    def put_artifact(self, job_id: str, local_path: str, artifact_name: Optional[str] = None) -> str:
        artifact_name = artifact_name or os.path.basename(local_path)
        artifact_dir = os.path.join(self.root, job_id, "artifacts")
        os.makedirs(artifact_dir, exist_ok=True)

        dst = os.path.join(artifact_dir, artifact_name)
        shutil.copy2(local_path, dst)
        return dst


store = LocalObjectStore()


def load_file_content(file_path: str) -> str:
    ext = os.path.splitext(file_path)[1].lower()

    if ext == ".csv":
        docs = CSVLoader(file_path=file_path).load()
        return "\n".join(d.page_content for d in docs[:50])

    if ext == ".pdf":
        docs = PyPDFLoader(file_path=file_path).load()
        return "\n".join(d.page_content for d in docs[:20])

    return "\n".join(d.page_content for d in TextLoader(file_path, encoding="utf-8").load())


class SandboxExecutor:
    def execute_python(self, code: str, mounted_input_file: str) -> Dict[str, Any]:
        import subprocess

        with tempfile.TemporaryDirectory() as td:
            script_path = os.path.join(td, "run.py")
            input_copy = os.path.join(td, os.path.basename(mounted_input_file))
            shutil.copy2(mounted_input_file, input_copy)

            with open(script_path, "w", encoding="utf-8") as f:
                f.write(code)

            result = subprocess.run(
                ["python", script_path],
                cwd=td,
                capture_output=True,
                text=True,
                timeout=60
            )

            generated_files = []
            for name in os.listdir(td):
                p = os.path.join(td, name)
                if os.path.isfile(p) and name not in {"run.py", os.path.basename(input_copy)}:
                    generated_files.append(p)

            return {
                "stdout": result.stdout,
                "stderr": result.stderr,
                "returncode": result.returncode,
                "generated_files": generated_files
            }


sandbox = SandboxExecutor()

RUNTIME_CONTEXT = {
    "job_id": None,
    "file_path": None,
}


@tool
def inspect_uploaded_file() -> str:
    """Inspect the uploaded file summary and return sample content."""
    if not RUNTIME_CONTEXT["file_path"]:
        return "No file available."
    text = load_file_content(RUNTIME_CONTEXT["file_path"])
    return text[:4000]


@tool
def execute_analysis_python(code: str) -> str:
    """Execute Python analysis code against the uploaded file in an isolated runtime."""
    if not RUNTIME_CONTEXT["file_path"]:
        return "No file available."

    result = sandbox.execute_python(code=code, mounted_input_file=RUNTIME_CONTEXT["file_path"])

    artifact_paths = []
    for generated in result["generated_files"]:
        saved = store.put_artifact(RUNTIME_CONTEXT["job_id"], generated)
        artifact_paths.append(saved)

    return json.dumps({
        "stdout": result["stdout"],
        "stderr": result["stderr"],
        "returncode": result["returncode"],
        "artifacts": artifact_paths
    }, ensure_ascii=False, indent=2)


@tool
def publish_report(markdown_text: str) -> str:
    """Persist a markdown summary as an output artifact."""
    if not RUNTIME_CONTEXT["job_id"]:
        return "No active job."

    with tempfile.TemporaryDirectory() as td:
        report_path = os.path.join(td, "analysis_report.md")
        with open(report_path, "w", encoding="utf-8") as f:
            f.write(markdown_text)

        saved = store.put_artifact(RUNTIME_CONTEXT["job_id"], report_path)
        return saved


def run_file_analysis(user_prompt: str, uploaded_file_path: str, original_name: str, mime_type: str) -> dict:
    stored = store.put_file(uploaded_file_path, original_name=original_name, mime_type=mime_type)
    RUNTIME_CONTEXT["job_id"] = stored.job_id
    RUNTIME_CONTEXT["file_path"] = stored.file_path

    model = ChatAnthropic(model="claude-sonnet-4-5")

    system_prompt = """
    Analysis assistant.
    If analysis needs computation, inspect the uploaded file first,
    then generate Python code and call execute_analysis_python.
    Always produce a final concise summary and publish a markdown report.
    """

    agent = create_agent(
        model=model,
        tools=[inspect_uploaded_file, execute_analysis_python, publish_report],
        system_prompt=system_prompt,
    )

    result = agent.invoke({
        "messages": [
            {"role": "user", "content": user_prompt}
        ]
    })
    return {
        "job_id": stored.job_id,
        "result": result
    }
```

This is a design sample based on the public patterns described in the LangChain docs and the open page. [\[bing.com\]](https://bing.com/search?q=LangGraph+persistence+docs), [\[reference....gchain.com\]](https://reference.langchain.com/python/langchain-community/document_loaders), [\[pypi.org\]](https://pypi.org/project/langchain-sandbox/), [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/langgraph/8.1-react-agent-%28create_react_agent%29)

# 4) LangGraph code sample

```python
from __future__ import annotations

import os
import uuid
import json
import shutil
import tempfile
from typing import TypedDict, List, Dict, Any

from langgraph.graph import StateGraph, START, END
from langgraph.checkpoint.memory import InMemorySaver


class AnalysisState(TypedDict, total=False):
    thread_id: str
    job_id: str
    input_file_path: str
    input_file_name: str
    mime_type: str
    user_request: str

    file_preview: str
    analysis_plan: str

    sandbox_stdout: str
    sandbox_stderr: str
    sandbox_returncode: int

    artifact_paths: List[str]
    final_summary: str
    status: str


OBJECT_ROOT = "./object_store_graph"
os.makedirs(OBJECT_ROOT, exist_ok=True)


def save_uploaded_file(src_path: str, original_name: str) -> tuple[str, str]:
    job_id = str(uuid.uuid4())
    job_dir = os.path.join(OBJECT_ROOT, job_id)
    os.makedirs(job_dir, exist_ok=True)

    dst = os.path.join(job_dir, original_name)
    shutil.copy2(src_path, dst)
    return job_id, dst


def preview_file(path: str) -> str:
    ext = os.path.splitext(path)[1].lower()
    if ext == ".csv":
        with open(path, "r", encoding="utf-8") as f:
            return "".join(f.readlines()[:20])
    with open(path, "r", encoding="utf-8", errors="ignore") as f:
        return f.read(3000)


def run_python_in_sandbox(input_file_path: str, code: str) -> Dict[str, Any]:
    import subprocess

    with tempfile.TemporaryDirectory() as td:
        script = os.path.join(td, "run.py")
        mounted = os.path.join(td, os.path.basename(input_file_path))
        shutil.copy2(input_file_path, mounted)

        with open(script, "w", encoding="utf-8") as f:
            f.write(code)

        result = subprocess.run(
            ["python", script],
            cwd=td,
            capture_output=True,
            text=True,
            timeout=60
        )

        artifacts = []
        for name in os.listdir(td):
            p = os.path.join(td, name)
            if os.path.isfile(p) and name not in {"run.py", os.path.basename(mounted)}:
                artifacts.append(p)

        return {
            "stdout": result.stdout,
            "stderr": result.stderr,
            "returncode": result.returncode,
            "artifacts": artifacts,
        }


def persist_artifacts(job_id: str, files: List[str]) -> Listout_dir = os.path.join(OBJECT_ROOT, job_id, "artifacts")
    os.makedirs(out_dir, exist_ok=True)

    persisted = []
    for f in files:
        dst = os.path.join(out_dir, os.path.basename(f))
        shutil.copy2(f, dst)
        persisted.append(dst)
    return persisted


def ingest_file(state: AnalysisState) -> AnalysisState:
    job_id, stored_path = save_uploaded_file(
        state["input_file_path"],
        state["input_file_name"]
    )
    return {
        "job_id": job_id,
        "input_file_path": stored_path,
        "status": "ingested"
    }


def inspect_file(state: AnalysisState) -> AnalysisState:
    preview = preview_file(state["input_file_path"])
    return {
        "file_preview": preview,
        "status": "inspected"
    }


def plan_analysis(state: AnalysisState) -> AnalysisState:
    plan = f"""
Analyze file: {state['input_file_name']}
Request: {state['user_request']}

Plan:
1. Read file
2. Generate summary statistics
3. Produce at least one chart
4. Save markdown summary
"""
    return {
        "analysis_plan": plan,
        "status": "planned"
    }


def execute_in_sandbox(state: AnalysisState) -> AnalysisState:
    ext = os.path.splitext(state["input_file_path"])[1].lower()

    if ext == ".csv":
        code = f"""
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("{os.path.basename(state['input_file_path'])}")
print(df.head())
print(df.describe(include='all'))

numeric_cols = df.select_dtypes(include='number').columns.tolist()
if numeric_cols:
    df[numeric_cols].hist(figsize=(10, 6))
    plt.tight_layout()
    plt.savefig("numeric_histograms.png")

with open("summary.md", "w", encoding="utf-8") as f:
    f.write("# Analysis Summary\\n\\n")
    f.write("Rows: " + str(len(df)) + "\\n\\n")
    f.write("Columns: " + ", ".join(df.columns) + "\\n")
"""
    else:
        code = f"""
with open("{os.path.basename(state['input_file_path'])}", "r", encoding="utf-8", errors="ignore") as f:
    text = f.read()

print(text[:2000])

with open("summary.md", "w", encoding="utf-8") as out:
    out.write("# Text Summary\\n\\n")
    out.write(text[:3000])
"""

    result = run_python_in_sandbox(state["input_file_path"], code)
    return {
        "sandbox_stdout": result["stdout"],
        "sandbox_stderr": result["stderr"],
        "sandbox_returncode": result["returncode"],
        "artifact_paths": result["artifacts"],
        "status": "executed"
    }


def persist_outputs(state: AnalysisState) -> AnalysisState:
    persisted = persist_artifacts(state["job_id"], state.get("artifact_paths", []))
    return {
        "artifact_paths": persisted,
        "status": "persisted"
    }


def summarize_results(state: AnalysisState) -> AnalysisState:
    summary = f"""
Execution status: {state.get('sandbox_returncode')}
Artifacts: {state.get('artifact_paths', [])}

STDOUT:
{state.get('sandbox_stdout', '')[:2000]}

STDERR:
{state.get('sandbox_stderr', '')[:1000]}
"""
    return {
        "final_summary": summary,
        "status": "completed"
    }


builder = StateGraph(AnalysisState)

builder.add_node("ingest_file", ingest_file)
builder.add_node("inspect_file", inspect_file)
builder.add_node("plan_analysis", plan_analysis)
builder.add_node("execute_in_sandbox", execute_in_sandbox)
builder.add_node("persist_outputs", persist_outputs)
builder.add_node("summarize_results", summarize_results)

builder.add_edge(START, "ingest_file")
builder.add_edge("ingest_file", "inspect_file")
builder.add_edge("inspect_file", "plan_analysis")
builder.add_edge("plan_analysis", "execute_in_sandbox")
builder.add_edge("execute_in_sandbox", "persist_outputs")
builder.add_edge("persist_outputs", "summarize_results")
builder.add_edge("summarize_results", END)

checkpointer = InMemorySaver()
graph = builder.compile(checkpointer=checkpointer)
```

This sample reflects the public LangGraph pattern of graph nodes plus checkpointed execution with a `thread_id`. # 5) When to choose which

## Prefer LangChain when

* tool use is enough
* workflows are short
* durable resume is not the main concern

## Prefer LangGraph when

* state must persist across turns
* processing is multi-stage
* resumability matters
* human-in-the-loop or fault tolerance is important

LangGraph persistence docs explicitly list memory, time travel, human-in-the-loop, and fault tolerance as reasons to use persistence. [\[deepwiki.com\]](https://deepwiki.com/langchain-ai/docs/2.2-langgraph-framework-documentation)

# 6) Production hardening checklist

The items below are **recommendations**, not direct quotes from the sources:

* use a **remote sandbox/container**, not host-bound execution
* enforce **CPU / memory / timeout limits**
* keep large files in storage; put only file identifiers into graph state
* store execution logs and artifact metadata separately
* return artifact paths/URLs instead of embedding large binary outputs in agent state
* add queue/worker execution for long-running jobs
* validate content type before analysis

These recommendations are consistent with the public sandbox documentation’s security motivation and the workflow shape described on the source page. [\[pypi.org\]](https://pypi.org/project/langchain-sandbox/), [\[bing.com\]](https://bing.com/search?q=LangGraph+persistence+docs)
