## *Enhancements with MCP Servers*

Source: [Anthropic Course Page – Enhancements with MCP servers](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287792)

***

# 1. 🔍 Summary

This page explains how **Model Context Protocol (MCP)** enhances Claude Code by allowing it to connect to external tools and services via **MCP servers**.

### Key concepts:

* Claude Code includes a **built-in MCP client**
* MCP enables extension beyond default capabilities by connecting to external systems
* MCP servers expose three components:
  * **Tools** → perform actions
  * **Resources** → provide data
  * **Prompts** → reusable templates
* MCP servers are registered via CLI:
  ```bash
  claude mcp add [server-name] [command]
  ```
* Example:
  ```bash
  claude mcp add documents uv run main.py
  ```
* Once connected, Claude can **automatically invoke tools**
* Real use case:
  * Convert documents (PDF/Word → Markdown) using a custom tool
* Ecosystem integrations:
  * Sentry → bug tracking
  * Playwright → browser automation
  * Figma → design context
  * Atlassian → Jira / Confluence
  * Slack → communication
  * Firecrawl → web scraping

👉 Main idea:  
MCP transforms Claude into a **workflow-aware agent** that interacts with real-world systems.
