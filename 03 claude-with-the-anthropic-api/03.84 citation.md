**Citations**
## 1. Summary

**Source:** [Citations – Claude API Course](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)

* **Citations feature** enables Claude to **show exactly where information comes from** in provided documents (e.g., PDF or text). [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
* Solves the **transparency problem**: instead of appearing to rely on training data, responses are **grounded in source documents**. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
* Requires adding `citations: { enabled: True }` to the document input structure. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
* Output becomes **structured with citation metadata**, including:
  * `cited_text`
  * `document_index`
  * `document_title`
  * `start_page_number`, `end_page_number` [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
* Works for **PDF (page-based)** and **plain text (character-based)** sources. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
* Enables building **interactive UIs** where users can trace answers back to original content. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
* Key use cases:
  * Verification of accuracy
  * Working with authoritative documents
  * Improving user trust and explainability [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)

## 2. Q\&A

**Q1. What problem do citations solve?**  
A: They provide transparency by linking responses directly to source documents, allowing verification. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)

**Q2. How are citations enabled?**  
A: By adding `citations: { enabled: True }` inside the document block. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)

**Q3. What extra data is returned when citations are enabled?**  
A: Structured metadata including cited text, document index/title, and location (page or character position). [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)

**Q4. Does citation support only PDFs?**  
A: No, it works with both PDFs and plain text sources. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)

**Q5. Why is this important for enterprise systems?**  
A: It ensures traceability and builds trust by proving answers are grounded in real documents. [\[Citations\]](https://anthropic.skilljar.com/claude-with-the-anthropic-api/287771)
