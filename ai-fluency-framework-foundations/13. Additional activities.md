## 2. Key concepts

### 2.1 Purpose of this section

* Provides **self-paced, post-course activities** to deepen AI Fluency skills
* Focus: **practice-oriented learning** (not theory)
* Activities aligned with **4D framework** (Delegation, Description, Discernment, Diligence) [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

***

### 2.2 Build a Personal AI Fluency Plan

#### Competency Assessment

* Rate proficiency for each 4D:
  * Novice / Developing / Confident
* Identify:
  * Strengths
  * Gaps
* Evaluate usage of interaction modes:
  * Automation
  * Augmentation
  * Agency [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

#### Development Prioritization

* Select **1–2 competencies** to improve first
* Focus on:
  * Highest impact areas
  * Relevant use cases in current work
* Define target interaction modes to improve [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

#### Action Planning

* Define:
  * Specific practice activities
  * Timeline & frequency
  * Tools/resources
  * Success criteria (how progress is measured)
* Use AI (e.g., Claude) as a **thinking partner** for feedback [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

***

### 2.3 Build a Prompt & Pattern Library

#### Common Task Templates

* Identify **5–10 recurring AI tasks**
* Create reusable prompts with:
  * Structured format
  * Placeholders for dynamic inputs [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

#### Strategy Documentation

* Capture:
  * Effective **Description techniques**
  * Useful **Discernment patterns**
  * Best approaches per task type / interaction mode [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

#### Reference System

* Organize for:
  * Searchability
  * Reuse
* Maintain:
  * Example outputs
  * Continuous updates based on new learnings [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

***

### 2.4 Practice via Games (Core Insight)

#### Why games?

* Structured tasks → improve **Description precision**
* Ambiguity/trickiness → improve **Discernment**
* Encourages **interactive reasoning with AI** [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

***

### 2.5 Example Activities

#### Riddle Exchange

* Focus:
  * AI reasoning explanation
  * Guiding AI without giving answer
* Skill:
  * Steering chain-of-thought

#### Cooperative Crosswords

* Practice:
  * Constraint-based reasoning
  * Iterative refinement
* Example guidance:
  * Narrowing possibilities
  * Reframing meaning [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

#### Word Association

* Identify:
  * Weak reasoning
  * Missed patterns
* Improve collaborative discovery

***

### 2.6 Discernment Practice Types

#### Product Discernment

* Validate output correctness and coherence

#### Process Discernment

* Evaluate how AI reasons:
  * Multiple possibilities?
  * Bias toward one interpretation?

#### Performance Discernment

* Assess communication quality:
  * Clarity
  * Questioning
  * Collaboration quality [\[Additional...activities\]](https://anthropic.skilljar.com/ai-fluency-framework-foundations/314863)

***

### 2.7 Additional Practice Activities

* **Twenty Questions**

## 3. Q\&A

**Q1. What is the main goal of this section?**

* Reinforce AI Fluency through hands-on, self-directed practice

**Q2. Why create a personal AI fluency plan?**

* To systematically improve targeted competencies with measurable progress

**Q3. What is a prompt & pattern library?**

* A reusable collection of structured prompts and proven strategies

**Q4. Why are games used?**

* They provide controlled environments to train:
  * Clear instruction (Description)
  * Critical evaluation (Discernment)

**Q5. What are the three types of Discernment?**

* Product (output quality)
* Process (reasoning approach)
* Performance (communication effectiveness)
