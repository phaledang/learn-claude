# Connecting Your Tools

## Summary
Source: https://anthropic.skilljar.com/claude-101/383397

- Connectors allow Claude to access real tools and data
- Transform Claude into an informed collaborator
- Enable reading, searching, analyzing, creating, and executing tasks

### MCP (Model Context Protocol)
- Standard interface like "USB-C for AI"
- Enables seamless integration across tools

## Connector Types
- Web connectors: Google Drive, Slack, Notion, Asana
- Desktop extensions: Local files, apps, browser automation

## Setup Flow
1. Open connector directory
2. Click Connect
3. Authenticate
4. Grant permissions
5. Test connection

## Use Cases
- Project management: create, summarize tasks
- Communication: draft replies, search threads
- Documentation: search and summarize
- Business tools: revenue, transactions, CRM

## Security
- Scoped permissions
- User-level access only
- Revocable anytime
- Install from trusted sources

## Q&A
- What are connectors? Tools for integrating Claude with systems
- Why important? Remove manual copy-paste and add context
- What is MCP? Standard protocol for integrations
- Types? Web and desktop
- Setup? Connect → Authenticate → Permissions → Test
