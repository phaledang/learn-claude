## 1. Summary of the page

**Page:** *Custom commands – Claude Code in Action*

### Core concept

* Claude Code supports **custom commands** to automate repetitive developer workflows. [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)
* Commands are invoked using `/command-name` (similar to built-in slash commands). [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)

### How to define custom commands

* Setup structure:
  * Create `.claude/commands/` directory
  * Add a Markdown file (e.g., `audit.md`)
  * Filename becomes command name → `/audit` [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)
* Commands are **auto-loaded** (no restart required). [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)

### Example: Dependency audit command

* A command can include multiple steps:
  * `npm audit`
  * `npm audit fix`
  * run tests afterward  
    → enables **multi-step workflow automation** in one command [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)

### Arguments support

* Use placeholder: `$ARGUMENTS`
* Example:
  ```
  /write_tests path/to/file
  ```
* Arguments can be:
  * file paths
  * any free-form text for context [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)

### Benefits

* Automation of repetitive tasks
* Consistent execution of workflows
* Context injection for Claude
* Flexible reuse across inputs [\[Custom commands\]](https://anthropic.skilljar.com/claude-code-in-action/303234)

## 4. Key takeaways

* Custom commands = **prompt-based automation layer**
* Structure:
  ```
  .claude/
    commands/
      my-command.md
  ```
* Command = reusable workflow + instruction template
* `$ARGUMENTS` → parameterization mechanism
* Enables:
  * standardization
  * reduced cognitive load
  * repeatable developer workflows
* Complements (not replaces):
  * MCP tools (execution layer)
  * agents/workflows (orchestration layer)
