## 1. Summary (from active page)

**Page:** Claude Code setup  
**URL:** 

https://anthropic.skilljar.com/claude-code-in-action/301614

[Claude Code Quickstart](https://code.claude.com/docs/en/quickstart)

**Purpose**

* Guide to **install and initialize Claude Code locally**
* Entry step before using Claude Code features (coding assistant, MCP, hooks, etc.)

**Core steps**

* Install Claude Code (multiple OS supported)
* Run CLI (`claude`)
* Authenticate with Claude account
* Configure environment (theme, credentials)
* Handle installation issues

**Installation options**

* **Mac / Linux / WSL**
  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```
* **Windows PowerShell**
  ```powershell
  irm https://claude.ai/install.ps1 | iex
  ```
* **Windows CMD**
  ```cmd
  curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
  ```
* **Mac (Homebrew)**
  ```bash
  brew install --cask claude-code
  ```

**Post-install**

* Run:
  ```bash
  claude
  ```
* First run:
  * Select terminal theme
  * Authenticate with `claude.ai`

**Troubleshooting**

* Issues may include:
  * CLI not found
  * Network errors
  * Permission issues
* Refer to official docs troubleshooting section

**Cloud provider note**

* Additional setup required if using:
  * Amazon Bedrock
  * Google Vertex AI
  * Microsoft Foundry
