## 1. Summary of the page

**Page:** <https://anthropic.skilljar.com/claude-code-in-action/301615>

### Project Setup – Claude Code in Action

* Purpose: provide a **hands-on project environment** to explore Claude Code capabilities using a sample UI generation app. [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)

### Setup requirements

* Install **Node.js locally** [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)
* Download project package: `uigen.zip` [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)
* Extract and navigate to project directory [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)

### Initialization steps

* Run:
  ```bash
  npm run setup
  ```
  * Installs dependencies
  * Sets up local SQLite database [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)

### Optional: Enable Claude API

* Used for **UI component generation via Anthropic API** [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)
* Steps:
  * Get API key from Anthropic console
  * Add to `.env`:
    ```
    your-api-key-here
    ```
  * Without API key → system generates **mock/static code** [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)

### Run project

```bash
npm run dev
```

* Starts development server [\[Project setup\]](https://anthropic.skilljar.com/claude-code-in-action/301615)
