# 2. Key Concepts

## Overview

Claude Cowork is designed for delegated work rather than conversational assistance. The lesson focuses on:

* Identifying tasks that are a good fit for Cowork.
* Using Scheduled Tasks for recurring work.
* Using Dispatch to start tasks from a mobile device.
* Understanding when to hand work off to Claude and when human judgment remains necessary. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

## Learning Objectives

By the end of the lesson:

* Recognize the three work patterns that fit Cowork.
* Create recurring workflows using `/schedule`.

## Three Patterns of Work That Suit Cowork

### Pattern 1: Multi-Step Tasks

Cowork works best when a task requires multiple actions.

Instead of manually managing each step:

1. Gather context
2. Compare information
3. Research
4. Draft output
5. Format deliverable

Cowork can execute the full workflow from a single prompt. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Examples

* Organize a week of customer feedback emails into themes.
* Extract figures from reports and spreadsheets.
* Build a dashboard from multiple data sources. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### What Happens Internally

Cowork may:

1. Read source files
2. Compare information
3. Draft deliverables
4. Format output
5. Perform self-review

Progress is visible through the Progress Panel. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

***

### Pattern 2: File-Based Tasks

Good fit when:

* Inputs are real files.
* Output is a real artifact.
* Existing files need modification.

Examples of outputs:

* Word documents
* PowerPoint decks
* PDFs
* Spreadsheets [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Difference from Chat

**Claude Chat**

* Creates new content.

**Claude Cowork**

* Reads existing files.
* Edits files.
* Saves updates back into the workflow. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Examples

* Generate a client proposal from templates and meeting notes.
* Create monthly reports from spreadsheets including charts. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

***

### Pattern 3: Multi-Tool Tasks

Best when work spans several systems.

Examples:

* Gmail
* Slack
* Microsoft 365
* Calendar
* CRM systems [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Difference from Chat

Chat:

* Works with tools individually.

Cowork:

* Plans across multiple tools.
* Executes an entire workflow as a single delegation. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Examples

* Draft Outlook follow-up emails using calendar and meeting notes.
* Search Slack discussions and generate launch summaries. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

## Scheduled Tasks

### Purpose

Automate recurring work. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Method 1

Start from scratch:

```text
/schedule
```

Then describe:

* The task
* The schedule/cadence

Claude drafts a prompt for approval. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Method 2 (Recommended)

1. Perform task in Cowork.
2. Verify result.
3. Run `/schedule`.
4. Convert workflow into a recurring task. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Supported Cadences

* Hourly
* Daily
* Weekdays
* Manual [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Important Requirement

Scheduled tasks only run when:

* Computer is on.
* Claude is running. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

If the machine is asleep or off:

* Task is delayed.
* Claude resumes when the machine becomes available. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Scheduling Ideas

#### Weekly Review

* Weekly deliverables shipped
* Major decisions from Slack or Teams
* Next week's priorities

#### Monthly Metrics

* Spreadsheet aggregation
* Metrics reporting

#### Morning Briefing

* Today's meetings
* Related meeting notes
* Email thread context [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

## Dispatch

### Purpose

Start Cowork tasks from a mobile device. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### How It Works

* Create task on phone.
* Work executes on desktop.
* Uses desktop files.
* Uses desktop connectors.
* Uses desktop permissions.
* Notification arrives when completed. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Requirements

Desktop must be:

* On
* Awake
* Signed in
* Running Cowork [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Availability

Dispatch is:

* Research Preview
* Available on Pro and Max plans

Enterprise availability depends on:

* Organization rollout
* IT administrator configuration [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

## Core Idea

The three indicators that a task should be delegated:

| Indicator  | Meaning                 |
| ---------- | ----------------------- |
| Multi-step | Requires many actions   |
| File-based | Reads/writes real files |
| Multi-tool | Uses multiple systems   |

When a task matches one or more of these patterns, it is a strong candidate for Claude Cowork. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

# 3. Q\&A

### Q1. What types of work are ideal for Claude Cowork?

Work that is:

* Multi-step
* File-based
* Multi-tool [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Q2. What is the biggest difference between Chat and Cowork?

Chat helps interactively task-by-task, while Cowork executes an entire delegated workflow across files and tools. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Q3. How do scheduled tasks get created?

Use:

```text
/schedule
```

Either on a new task or after successfully completing a task once. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Q4. When do scheduled tasks run?

Only when the computer is powered on and Claude is running. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Q5. What is Dispatch?

A capability that lets users assign Cowork tasks from the Claude mobile app while execution happens on the desktop. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Q6. Does Dispatch run if the desktop is off?

No. The task waits until the desktop becomes available. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)

### Q7. Who is responsible for final decisions?

The human user. Claude assists while human judgment remains in control. [\[What Claud...do for you\]](https://anthropic.skilljar.com/introduction-to-claude-cowork/444169)
