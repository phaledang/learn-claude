# Claude Code 101 – Your First Prompt

## Summary
Source: https://anthropic.skilljar.com/claude-code-101/469791

- Interact with Claude Code using natural prompts
- Be descriptive to improve output quality

### Modes
- Approval mode
  - Ask permission before edits and commands
- Auto-accept mode
  - Auto-approve edits
  - Commands still require approval
- Switch using Shift + Tab

### Plan Mode
- Read-only analysis of codebase
- Generates detailed execution plan
- Asks clarifying questions
- Best for:
  - Complex tasks
  - Multi-step implementations
  - Code reviews

### Workflow example
1. Run claude
2. Use Plan Mode
3. Write prompt
4. Review plan
5. Approve execution
