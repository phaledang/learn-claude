

# 🧠 1. Summary

**Page:** [Subagents – Claude Code 101](https://anthropic.skilljar.com/claude-code-101/469796)

## Core concept

* **Subagents = lightweight delegated agents** that Claude uses to:
  * Break tasks into smaller parts
  * Execute them **in parallel**
  * Keep main context clean

## Why subagents matter

* Claude’s **context window is limited**
* Exploration tasks (code search, web search, tool calls) consume context
* Not all intermediate steps are relevant to final output

👉 Subagents solve this by:

* Running in **separate isolated context windows**
* Doing heavy work (e.g., explore codebase)
* Returning **only summarized results** to main agent

## How it works

* Claude creates a subagent for a task (e.g.,*“explore this repo”*)
* Subagent:
  * Runs independently
  * Performs exploration / analysis
  * Summarizes findings
* Main Claude:
  * Receives only final summary
  * Keeps context clean

## Creation flow

* Use command:
  ```
  /agents
  ```
* Then:
  * Create new agent
  * Define:
    * Purpose
    * Scope
    * Tools
    * Prompt
* Claude auto-generates:
  * Name
  * Description
  * Trigger logic

## Customization capabilities

* **Persistent memory**
  * Subagent remembers across conversations
* **Preloaded skills**
  * Add skill keys → full skills loaded into context
* More control over behavior and reuse

## Key takeaway

* Subagents = **best practice for context management**
* Helps:
  * Reduce noise
  * Improve performance
  * Scale complex tasks

# ❓ 2. Q\&A

### Q1: What is a subagent?

A: A delegated agent that performs tasks independently and returns summarized results.

### Q2: Why are subagents important in Claude Code?

A: They prevent context overload by isolating heavy computation in separate context windows. [\[Subagents\]](https://anthropic.skilljar.com/claude-code-101/469796)

### Q3: How do subagents improve performance?

A: By running tasks in parallel and returning only summarized outputs.

### Q4: What problem do subagents solve?

A: Context window pollution from irrelevant intermediate steps.

### Q5: How do you create a subagent?

A: Use `/agents` and follow guided setup steps.

### Q6: Can subagents retain memory?

A: Yes, via persistent memory across conversations. [\[Subagents\]](https://anthropic.skilljar.com/claude-code-101/469796)

### Q7: What is the benefit of preloading skills?

A: Subagent gets full skill context, improving task execution.
