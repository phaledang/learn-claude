# What is Claude Code?

## Source
https://anthropic.skilljar.com/claude-code-101/469788

## Summary

### Core idea
- Claude Code is an agentic coding tool
- It understands your codebase
- It can edit files, run commands, and integrate with developer tools

### Key differences vs Claude
- Claude chat requires copy/paste
- Claude Code directly operates on your environment
- Works as an AI Agent

### AI Agent concept
- Operates in a loop (observe → think → act)
- Can interact with tools, services, and systems

### Capabilities
- Read and understand codebase
- Edit and refactor files
- Run terminal commands
- Search web for documentation

### Important concepts
- Context window (limited working memory)
- Permission-based execution
- Not perfect → requires human validation

## Q&A

**Q: What is Claude Code?**  
A: An agentic coding tool that works directly on your codebase.

**Q: How is it different from Claude chat?**  
A: It can interact with files and systems directly instead of relying on copy/paste.

**Q: What is an AI Agent?**  
A: Software that performs actions in an environment to achieve a goal.

**Q: What can it do?**  
A: Analyze, edit, run commands, and search documentation.

**Q: What is the limitation?**  
A: Context window limits and possible mistakes.
