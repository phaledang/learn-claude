# How Claude Code works

Source: https://anthropic.skilljar.com/claude-code-101/469789

---

## Summary
Claude Code is an agent-based coding assistant.

Key concepts:
- Agentic loop: prompt → context gathering → action → verification → repeat
- Context window: stores conversation, files, outputs
- Automatic compaction when limits reached
- Tools: enable actions (file read, commands, search)
- Permissions: control execution safety

---

## Q&A

**Q: What makes Claude Code different from chat AI?**  
A: It can execute actions and verify results.

**Q: What is the agentic loop?**  
A: A cycle of understanding, acting, verifying, and repeating.

**Q: What is the context window?**  
A: A limit on how much information Claude can retain.

**Q: What happens when context is full?**  
A: Claude compacts/summarizes the conversation.

**Q: What do tools do?**  
A: They allow Claude to perform actions.

**Q: What permission modes exist?**  
A: Default, Auto-accept, Plan mode.
