# Your first conversation with Claude

## Summary
Page: https://anthropic.skilljar.com/claude-101/383390

This lesson explains how to start working with Claude effectively. Claude is treated as a collaborative AI partner. Good prompts should include context, task, and rules. Iteration and feedback improve results, and additional context can be provided via file uploads. Claude also supports memory and styles for personalization.

---

## Q&A

**Q1: How should you talk to Claude?**  
A: Like a coworker—naturally and conversationally.

**Q2: What are the 3 parts of a good prompt?**  
A: Setting the stage, defining the task, specifying rules.

**Q3: Why is context important?**  
A: It improves response accuracy.

**Q4: How to improve responses?**  
A: Follow-up questions, feedback, refine prompts.

**Q5: What files can Claude process?**  
A: PDF, DOCX, CSV, TXT, PNG, JPEG.
