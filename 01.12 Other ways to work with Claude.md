## Summary

Claude is not just claude.ai. It can be used across multiple tools depending on workflow.

### Tools

#### Claude Code
- Works in terminal, IDE, browser, Slack
- Write code from natural language
- Debug issues
- Understand codebase
- Automate dev tasks

#### Claude in Slack
- Works in Slack workspace
- Summarize threads
- Draft replies
- Prepare meetings
- Answer questions with context

#### Claude for Excel
- Works in Excel sidebar
- Analyze formulas
- Fix spreadsheet errors
- Create/update sheets
- Build charts

#### Claude for PowerPoint
- Works in PowerPoint sidebar
- Create slides from notes
- Rewrite slides
- Restructure decks
- Standardize formatting

#### Claude for Chrome
- Works as Chrome extension
- Summarize web pages
- Draft emails
- Automate browsing
- Maintain cross-tab context
- Note: Research preview
