# 📌 1. Summary

## 🔹 What are Skills?
Skills are **folders of instructions, scripts, and resources** that Claude dynamically loads to improve performance on specialized tasks.

They function like **expertise packages**, allowing Claude to:
- Execute tasks consistently
- Follow predefined workflows
- Automate repeatable processes

Examples:
- Creating Excel spreadsheets  
- Generating PowerPoint presentations  
- Producing PDF reports  

---

## 🔹 Types of Skills

### 1. Anthropic Skills
- Built and maintained by Anthropic  
- Automatically invoked by Claude  
- No manual setup required  
- Focus on document creation (Excel, Word, PowerPoint, PDF)

### 2. Custom Skills
- Created by users or organizations  
- Designed for domain-specific workflows  
- Examples:
  - Applying brand guidelines
  - Structuring meeting notes
  - Executing data analysis workflows
  - Running compliance checklists  

---

## 🔹 Enabling Skills

### Requirements
- Code execution must be enabled  
- File creation must be enabled  

### Steps
1. Go to **Settings → Capabilities**
2. Enable:
   - Code execution
   - File creation
3. Navigate to **Skills section**
4. Toggle Skills on/off

### Plan Notes
- Available for Pro, Max, Team, Enterprise plans  
- Enterprise requires admin enablement  
- Team plans: enabled by default  

---

## 🔹 Using Skills

Claude automatically selects Skills based on your prompt.

### Example prompts:
- "Create an Excel spreadsheet tracking monthly expenses"
- "Turn meeting notes into a PowerPoint presentation"
- "Generate a PDF report summarizing this data"
- "Build a financial model in Excel"

### Output:
- Downloadable file  
- Option to save to Google Drive  

---

## 🔹 File Execution

Claude can:
- Work with uploaded files (.xlsx, .pptx, .docx, .pdf)
- Perform analysis and updates
- Generate new versions of files

⚠️ Important:
- Claude does **not overwrite original files**
- It creates updated copies instead

### Requirement
- Enable **limited network access**

---

## 🔹 Security Considerations

Since Skills may contain executable code:

- Only install Skills from trusted sources
- Review custom Skills before using them
- Built-in Skills are tested by Anthropic
- Custom Skills are private to your account

---

## 🔹 Creating Custom Skills

Custom Skills are created conversationally with Claude.

### Steps:
1. Start a new chat  
2. Describe the skill  
   - Example: "Create a quarterly business review skill"
3. Answer Claude’s questions:
   - What should it do?
   - What defines good output?
   - When should it be used?
4. Upload references:
   - Templates
   - Brand guidelines
   - Example outputs  
5. Save the generated skill file  

After creation:
- The Skill appears in your **Skills list**
- Claude invokes it automatically when relevant  

### Iteration
- Skills can be improved over time  
- Claude can edit existing Skills  

---

## 🔹 Skills vs Projects

| Aspect | Projects | Skills |
|--------|--------|--------|
| Purpose | Store knowledge | Execute processes |
| Role | "What" (data/context) | "How" (workflow) |
| Use Case | Research, documents, notes | Repeatable workflows |
| Persistence | Available across chats | Triggered when needed |

### Key Insight
- **Projects = knowledge base**
- **Skills = execution engine**

### Example Combination
- Project: customer data repository  
- Skill: customer call preparation workflow

# ❓ 2. Q&A

**Q1: What are Skills?**  
A: Reusable instruction sets that help Claude perform tasks consistently.

**Q2: What is the difference between Anthropic and Custom Skills?**  
A:
- Anthropic Skills are built-in and automatic  
- Custom Skills are user-defined workflows  

**Q3: Do I need to manually trigger Skills?**  
A: No, Claude automatically selects them.

**Q4: What is required to enable Skills?**  
A: Code execution and file creation must be enabled.

**Q5: Can Claude edit existing files?**  
A: It creates updated versions instead of modifying originals.

**Q6: How do I create a custom Skill?**  
A: By describing your workflow in chat and letting Claude generate it.

**Q7: Are Skills safe?**  
A: Yes, if you use trusted sources and review custom Skills.

**Q8: How are Skills different from Projects?**  
A: Projects store knowledge, Skills define execution processes.

# 🔑 4. Key Takeaways

- Skills act as **execution layer** for Claude  
- Enable **repeatable and consistent workflows**  
- Two types:
  - Built-in (Anthropic)
  - Custom (user-defined)
- Claude automatically selects Skills  
- Custom Skills are powerful for enterprise use cases  
- Best practice:
  - Combine **Projects (context)** + **Skills (execution)**  
- Security is critical when using custom Skills
