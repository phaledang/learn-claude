# Working with Skills (Claude 101)

## Summary
Skills are reusable instruction sets that enhance Claude’s ability to perform specialized tasks.  
https://anthropic.skilljar.com/claude-101/383396  

### Key Concepts
- Skills = expertise packages (instructions + scripts + resources)
- Automatically selected by Claude
- Used for document creation and workflow automation

### Types
- **Anthropic Skills**: Built-in, automatic
- **Custom Skills**: User-defined workflows

### Enabling
- Requires:
  - Code execution
  - File creation
- Configure in Settings → Capabilities

### Usage
- Triggered automatically based on prompts
- Outputs files like Excel, PPT, PDF

### File Handling
- Works on uploaded files
- Creates new versions (no overwrite)

### Security
- Review custom Skills
- Use trusted sources

### Creating Skills
1. Describe requirements
2. Answer Claude questions
3. Provide examples
4. Save generated Skill

### Skills vs Projects
- Projects = knowledge
- Skills = execution
