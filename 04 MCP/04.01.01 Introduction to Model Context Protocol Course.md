## 4. Key Takeaways

### MCP Concept

* MCP = **standardized integration layer**
* Enables:
  * Tool use
  * Context sharing
  * Structured interaction

### Architecture

* **Server**
  * Defines:
    * Tools
    * Resources
    * Prompts
* **Client**
  * Executes interactions
  * Connects to server

### Development Pattern

* Define tools → expose via MCP server
* Build client → call tools
* Extend with:
  * Resources (data)
  * Prompts (reusable instruction)

### Why MCP matters

* Avoid custom integrations
* Provide reusable, modular AI systems
* Improve scalability of agent/tool ecosystems
