  * Roots (with walkthrough)
  * Transports and communication

***

### MCP Advanced Topics (High-level)

* Focus areas of the course:
  * **Agent–tool interaction patterns** (via sampling and workflow execution)
  * **Observability & feedback loops** (notifications, logs, progress)
  * **Context management** (roots, state)
  * **Transport layers** (STDIO, HTTP streaming)
  * **Protocol structure** (JSON-based message types) [\[Let's get started!\]](https://anthropic.skilljar.com/model-context-protocol-advanced-topics/296349)

***

### Learning Flow

* Start with introduction → progressively move into:
  1. Core protocol features
  2. Execution behavior (Sampling, Roots)
  3. Communication mechanisms (JSON + transports)
  4. Advanced runtime behavior (StreamableHTTP + state)
  5. Validation via assessment [\[Let's get started!\]](https://anthropic.skilljar.com/model-context-protocol-advanced-topics/296349)
