## 2. Key concepts

### Overview

* **Logging & progress notifications (MCP)**
  * Mechanism to provide **real-time feedback during tool execution** [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)
  * Critical for **long-running tasks** (research, data processing) [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)
  * Purpose: improve **UX, visibility, and trust**

***

### Problem solved

* **Without notifications**
  * No output until completion → unclear state (running vs stuck) [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)

* **With notifications**
  * Continuous feedback:
    * Status messages
    * Logs
    * Progress updates [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)

***

### MCP Server-side implementation

#### Context object

* Automatically injected into tool functions [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)
* Acts as **communication channel (server → client)**

#### Core APIs

* `context.info(message)`
  * Send **log messages** [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)

* `context.report_progress(current, total)`
  * Send **progress updates** [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)

***

### Example (Python MCP SDK)

```python
@mcp.tool(name="research", description="Research a given topic")
async def research(topic: str, *, context: Context):
    await context.info("About to do research...")
    await context.report_progress(20, 100)

    sources = await do_research(topic)

    await context.info("Writing report...")
    await context.report_progress(70, 100)

    results = await generate_report(sources)
    return results
```

* Flow:
  * Emit log → emit progress → execute → update → return result [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)

### MCP Client-side implementation

#### Logging callback

* Handles log messages from server

```python
async def logging_callback(params):
    print(params.data)
```

#### Progress callback

* Handles progress updates

```python
async def print_progress_callback(progress, total, message):
    if total is not None:
        percentage = (progress / total) * 100
        print(f"Progress: {percentage:.1f}%")
```

#### Attach callbacks

```python
async with ClientSession(read, write, logging_callback=logging_callback) as session:
    await session.initialize()
    await session.call_tool(
        name="add",
        arguments={"a": 1, "b": 3},
        progress_callback=print_progress_callback,
    )
```

* Client decides how to display notifications [\[learn-clau...arn-claude\]](https://github.com/phaledang/learn-claude/blob/main/model-context-protocol-advanced-topics/05.Log%20and%20progress%20notifications.md)
