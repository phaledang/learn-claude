# Enterprise Search - Training Notes

## 🔗 Source
[Enterprise Search Lesson](https://anthropic.skilljar.com/claude-101/383398)

---

## 🧠 Summary
- Enterprise Search = org-wide AI search across tools
- Adds "Ask {Org Name}" in sidebar
- Acts like a centralized knowledge base
- Connectors:
  - Documents: SharePoint, Google Drive
  - Chat: Slack, Teams
  - Email: optional

### Capabilities
- Cross-source querying
- Context-aware synthesis
- Source citations included

### Use Cases
- Status updates
- Policy lookup
- Research & analysis
- Onboarding
- Project tracking

---

## ⚙️ Setup

### Admin
- Enable Enterprise Search
- Connect tools
- Configure project name + description

### User
- Open “Ask {Org}”
- Authenticate connectors
- Start querying

---

## 🔐 Security
- Permission-based access
- No additional data indexing
- Private conversations

---

## ❓ Q&A

**Q: What is Enterprise Search?**  
A: Organization-wide knowledge search feature.

**Q: Difference from normal chat?**  
A: Designed for multi-source retrieval.

**Q: Required setup?**  
A: Admin first, then user authentication.

**Q: Is it secure?**  
A: Yes, respects existing permissions.
